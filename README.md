# 批量导入Shell

从 CSV 中 批量导入 Shell

> **已经修复原版导入数据无法正常双击打开的问题！**

## Example

```plain
CSV格式：
备注,分类,密码,类型,Shell地址

举个例子：
测试1,test,pass,asp,http://xxxx.com/a.asp
测试2,test,pass,jsp,http://xxxx.com/a.jsp
测试3,test,pass,php,http://xxxx.com/a.php
```

**`分类`、`密码`、`类型`、`Shell地址`必须要填写！！！**

如果备注不想填写就留空，如下：

```plain
,test,pass,php,http://gov.cn/666.php
```
